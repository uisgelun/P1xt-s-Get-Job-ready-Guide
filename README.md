##YDKJS - Up & Going
- - -

JS uses typed values, not variable types.

Built-in types:
* `string`
* `number`
* `boolean`
* `null` and `undefined`
* `object`
* `symbol`

The `typeof` operator returns the type of a JS variable.

For example:

`var a = "Hello world!";  
typeof a;        //"string"`
